# NASA Near Earth Object high scores!

This website will get you to find the fastest, closest, and most traveled NEOs! 
Start from a date, any date! 
See a list of NEOs and pick your favourite. 
See where that NEO goes, select the next time it's near earth, see a list of NEOs and pick a new one!


Repeat over and over and the website will record your stats.
